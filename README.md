# Vaibhav Petkar - Tech CEO & Full-Stack Developer Portfolio

Welcome to the source code for my personal portfolio and professional services website. This platform serves as a central hub for my software development projects, ERP systems, mobile applications, and online programming courses.

![Portfolio Preview](./public/vaibhav-avatar.png)

## 🌐 Live Website
**[https://vaibhavpetkar.site/](https://vaibhavpetkar.site/)**

## 🚀 Key Features

*   **Modern, Premium UI/UX:** Built with React, Tailwind CSS v4, and dynamic Framer Motion animations (mesh gradients, floating objects, typing effects, scroll reveals) to provide a "CEO-level" aesthetic.
*   **Comprehensive Services:** Details freelance offerings ranging from web/app development and Frappe/ERPNext customization to dedicated 1-on-1 programming mentorship.
*   **Course Registration System:** Integrated Direct-to-WhatsApp messaging flow for student onboarding across 20+ technologies (C++, Java, Python, Next.js, etc.).
*   **Robust SEO Configuration:** Complete structured JSON-LD Schema integration (LocalBusiness, Person, Course), Sitemap generation, Open Graph optimization, and Twitter Card support for maximum search engine dominance.
*   **Fully Responsive:** Fluid scaling across all mobile, tablet, and high-DPI desktop viewports.

## 🛠️ Technology Stack

*   **Frontend Framework:** React 19 + TypeScript
*   **Build Tool:** Vite
*   **Styling:** Tailwind CSS v4, Radix UI Primitives 
*   **Animations:** Framer Motion
*   **Routing & Forms:** Wouter, React Hook Form + Zod validation
*   **Icons:** Lucide React

## 📦 Local Deployment

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VaibhavPetkar/vaibhav-portfolio.git
   ```
2. **Install dependencies:**
   *(Note: This project uses complex dependencies, so `--legacy-peer-deps` may be required)*
   ```bash
   npm install --legacy-peer-deps
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
